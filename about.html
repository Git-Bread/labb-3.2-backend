Om uppgiften

Denna uppgift har varit trevlig, tog lite tid att få grepp om det hela och behövde läsa en hel del mongodb/mongoose documentation innan jag förstod hur allting fungerade (någorlunda).

Men hade som tur varit strukturerat api'n hyfsat, så behövde inte ändra särskillt mycket i denna del. Var bara id formatet som ändrades då mongodb vill ha ett id object (såvida jag inte vill ha auto) och formatera date objecten som lagras i den.

NoSql tycker jag hitils är mycket trevligare än sql, iaf att jobba med. Men som sagt i labbrapporten så fungerar de båda till olika syften, fixade fortfarande ingen server främst pågrund utav lathet i detta fall.
